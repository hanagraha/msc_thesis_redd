# REDD+ Thesis

## Description
This project is part of Hannah Graham's MSc thesis on comparing Global Forest Change and Tropical Moist Forest datasets for their suitability in REDD+ evaluation. 

## Usage
To reproduce results, first create the environment using the thesisenv.yaml file. All scripts are found in the scripts folder and should be run sequentially. In case of errors, the package rasterstats may need to be updated in the command line using code: pip install rasterstats --upgrade.
